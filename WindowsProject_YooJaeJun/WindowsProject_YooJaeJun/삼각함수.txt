삼각비
- 직각삼각형 두 변과 사잇각을 알 때 사용
비율: A와 B의 배수 관계

삼각함수: 세타를 인풋으로 넣고 아웃풋으로 삼각비를 갖는 함수

수학 -> sin cos tan
sinθ = b/c
cosθ = a/c
tanθ = b/a

a = c * cosθ
b = c * sinθ
-> 한 변의 길이를 알고 사이각을 안다면
떨어진 xy 좌표를 구할 수 있다.
c = a / cosθ = b / sinθ




도 => degree
공학용 도 => radian
원의 둘레 2파이알
원의 넓이 파이알제곱

호 == 반지름일 때 1 라디안
180 / 파이


// 호도법: 호의 길이로 각도를 나타내는 방법
호의 길이와 반지름의 길이가 같다면 1rad 이라고 한다.

360도 == 2ㅠ rad

1도 == ㅠ/180
1라디안 == 180/ㅠ

70도는 몇 라디안? 70 * ㅠ/180
